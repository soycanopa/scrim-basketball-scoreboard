# scrim-basketball-scoreboard
Este repositorio contiene el proyecto BasketballScoreboard para el Bootcamp de FrontEnd en Scrimba Repositorio con un tablero de puntuación de baloncesto en HTML, CSS y JS. Permite incrementar puntuaciones de dos equipos mediante botones, siguiendo un diseño específico.

El proyecto consiste en crear un tablero de puntuación de baloncesto desde cero, siguiendo un diseño específico proporcionado en un archivo Figma. El objetivo principal es desarrollar una aplicación interactiva que permita incrementar las puntuaciones de dos equipos (equipo local y visitante) al hacer clic en los botones correspondientes.

# Requisitos principales:

Incrementar la puntuación del equipo local o visitante según el valor del botón presionado.
El diseño debe seguir el archivo Figma proporcionado.
Implementar funcionalidad básica con HTML, CSS y JavaScript.
Objetivos adicionales (opcional):

Añadir un botón para reiniciar el juego (poner las puntuaciones a cero).
Resaltar el equipo que lleva ventaja.
Incluir más contadores (ej: periodo del partido, faltas, temporizador, etc.).
Personalizar el diseño según tus preferencias o adaptarlo a otro deporte.
